Conditional

image source: hackernoon.com

The if statement executes a statement if a specified condition is truthy. If the condition is falsy, another statement can be executed.

The operator that frequently used as a shortcut for the if statement is the conditional (ternary) operator.
This operator only takes three operands.

// example of ternary operator

let foo = true

let bar = foo ? 'foo' : 'bar'

Array

image source: homedit.com

Instead declare a lot of variable, you can store that item into a singel variable, on an Array.

Array in JavaScript can be nonhomogeneous, meaning the elements in an Array do not need to be the same type (it follows from this that arrays can have other arrays as elements).

Array containing another Array usually called as multidimension arrays.

let foo = [1, 2, 3, 4, 5] // array contain number

let bar = [[1, 2, 3], [4, 5, 6]] // array contain array

Loop de Loop

image source: yoyofactory-europe.com

Loops offer a quick and easy way to do something repeatedly.

There are many different kinds of loops, but they all essentially do the same thing:
they repeat an action some number of times (and it's actually possible that number could be zero).

The statements for loops provided in JavaScript are:

  • for statement

    • for (let i = 0; i < 5; i++) {

    console.log(i)

    }

  • while statement

    • let i = 0

    while (i < 5) {

    console.log(i)

    i++

    }

  • do...while statement

    • let i = 0

    do {

    console.log(i)

    i++

    } while (i < 5)
